# GitHub RAG Assistant

AI tool for analyzing GitHub repositories using RAG.

---

## Features
- Clone GitHub repos
- Code indexing with FAISS
- Language-aware chunking
- Chat with codebase using Gemini LLM
- Gradio UI

---

## Run with Docker

docker-compose up --build

---

## UI
http://localhost:7860

---

## Environment

GOOGLE_API_KEY=your_api_key
